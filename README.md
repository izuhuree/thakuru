# ތަކުރު (Thakuru Studio)

ދިވެހި ބަހަށް ޚާއްސަ ލިޔުންތެރިކަމުގެ އެހީތެރިއެކެވެ. މި އެޕްލިކޭޝަނަކީ ދިވެހި ލިޔުންތެރިންނަށް އެހީތެރިވުމަށްޓަކައި ޒަމާނީ އޭ.އައި (AI) ބޭނުންކޮށްގެން އުފައްދާފައިވާ ސްޓޫޑިއޯއެކެވެ.

## ކޮންފިގަރޭޝަން (Configuration)

- **Repository:** [https://github.com/izuhuree/thakuru](https://github.com/izuhuree/thakuru)
- **Version:** v5.0.0
- **Main Branch:** `main`
- **Creator:** އިޒޫ ކޮންސަލްޓަންސީ އެންޑް ޓްރެއިނިންގ

## މައިގަނޑު ބައިތައް (Core Features)

- **ހޮޅުއަށި:** ޚިޔާލުތައް ފުޅާކޮށް މަޝްވަރާކުރުމަށް.
- **އިސްލާހު:** ގަވާއިދާއި އިމްލާގެ ކުށްތައް ރަނގަޅުކުރުމަށް.
- **ސިޓީ:** ރަސްމީ ސިޓީތައް ފަސޭހައިން ތައްޔާރުކުރުމަށް.
- **ތަރުޖަމާ:** ދިވެހި އަދި އިނގިރޭސި ބަހުން ތަރުޖަމާކުރުމަށް.
